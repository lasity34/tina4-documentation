# Introduction

Installation

MacOS / Linux

```bash
python3 -m venv venv
source ./venv/bin/activate
pip install mkdocs
pip install --upgrade pip
```

Running
```bash
mkdocs serve
```
Building
```bash
mkdocs build
```
